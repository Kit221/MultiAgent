# MultiAgent

Мультиагентная система для анализа рынка труда IT-специалистов. Генерирует структурированный отчет (навыки, зарплаты, план обучения) на основе заданной роли.
Навыки: Выделяет Hard/Soft skills и тренды.
Зарплата: Оценивает вилки зарплат по грейдам и регионам.
Советник: Составляет план обучения и предлагает пет-проект.
Критик: Проверяет согласованность данных и выставляет оценку качества.

# Стек
- **Python 3.10+**
- **Ollama** (Local LLM)
- **Pydantic** (Валидация данных)

# Требования
1. Установлен и запущен [Ollama](https://ollama.com/).
2. Скачана модель: `ollama pull qwen2.5` (или укажите свою в `.env`).
